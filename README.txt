Ce projet a pour objectif de prendre en main et tester le framework css BootStrap.
Cette prise en main s'appuie sur le cours openclassroms https://openclassrooms.com/courses/prenez-en-main-bootstrap.
Le projet contient les répertoires css et js des versions compressées .min des fichiers du bootstrap.
A la racine, se situent les pages html des différents essais effectués.
La première page correspond au template de base donné dans le cours.
